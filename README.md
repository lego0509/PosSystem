# 学園祭カフェ POS システム

学園祭カフェのレジ〜キッチン〜受け渡し向けに作成したフロントエンドのみの簡易 POS システムです。バックエンドは持たず、ブラウザの `localStorage` に状態を保存します。

## 収録画面

- `/pos` : レジ担当者が注文を入力する 3 カラムレイアウト。
- `/kds` : キッチン側が調理進行を管理する 3 列ボード。
- `/call` : 完成済みの番号を大書表示する呼び出し画面。

トップページ（`/`）には各画面への導線を配置しています。

## 主な機能

### 共通
- 4 桁の注文番号を自動採番（0001〜9999）。
- data-testid 属性を主要コンポーネントに付与。
- レスポンシブ対応（タブレット横置き〜フル HD）。
- localStorage を疑似 API として利用。

### 注文入力 `/pos`
- F1〜F5 でカテゴリ切り替え、1〜9 で次回数量指定、Ctrl+Z で直前取消、Enter で注文確定。
- 商品カードはタッチ操作に最適化した大ボタン。無料オプションはモーダルで選択。
- カートでは無料オプションの要点表示、数量調整、行メモ入力が可能。
- 受付一時停止トグルと赤帯表示を実装。
- 注文確定後に 4 桁番号を大きく表示し案内。

### キッチンボード `/kds`
- 待ち / 調理中 / 完成 の 3 列にドラッグ＆ドロップまたはクリックで遷移。
- 無料オプションを色付きバッジで表現。経過タイマーを mm:ss で自動更新。
- 検索（番号・名前）とカテゴリフィルタを搭載。Undo ボタンで前の列に戻せます。
- 状態変更時に効果音を再生。

### 呼び出し `/call`
- 完成状態の番号を超特大グリッドで表示し、新規完成時に数秒ハイライト＆効果音。
- 番号のタップ／Enter／Space で再掲対象に選択。再掲ボタンで再度ハイライト。
- 受け渡し完了ボタンで番号を履歴へ移動。最近 10 件を下部に表示。
- 全画面表示ボタン（トグル）を備えています。

## 開発・動作方法

1. 依存パッケージは不要です（Node.js 18 以降推奨）。
2. ルートディレクトリで下記を実行してください。

   ```bash
   npm run dev
   ```

3. 表示された URL（既定: `http://localhost:4173`）へアクセスし、用途に応じて `/pos` `/kds` `/call` に遷移してください。

`localStorage` に保存されたデータを初期化したい場合は、ブラウザの開発者ツールから `school-fes-pos-state` を削除するか、`src/storage.js` の `clearAll()` をコンソールで呼び出してください。
